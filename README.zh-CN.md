## <span id="1">1、特性</span>
- 代码简洁并且易于定制化
- 通过多协程加速解析AOF文件

## <span id="2">2、安装</span>

### (1) Linux / Mac
```bash
git clone https://github.com/WGrape/parseAOF
```

### (2) Windows
暂不支持

## 3、Usage
执行 ```start.sh``` 脚本，并传入待解析的AOF文件路径

```bash
bash ./start.sh ./data/appendonly.aof
```
